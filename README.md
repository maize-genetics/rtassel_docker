# rTASSEL + Docker

Docker image for `rTASSEL`. Contains the following:
  + RStudio Server
  + CRAN packages:
    + `ape`
    + `cli`
    + `knitr`
    + `plotly`
    + `rlang`
    + `tidyverse`
    + `vctrs`
  + Bioconductor packages
    + `BiocStyle`
    + `AnnotationDbi`
    + `GenomicFeatures`
    + `GenomicRanges`
    + `IRanges`
    + `S4Vectors`
    + `SummarizedExperiment`
